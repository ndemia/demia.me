Context

Another of my failed interview test series. I applied for a position at a company I was dying to work for. This time around, I did manage to pass the technical assessment! With some good parts and some bad parts, of course. But in the end, I didn't pass the penultimate interview.

Tech stack

HTML, CSS (SASS), TypeScript.

Process

The assignment was meant to be completed in JavaScript, without any frameworks or tools, just plain vanilla. I've rebuilt it using TypeScript to improve it wherever I could, and to get more familiar with it.

Problems

It was my first time dealing with pagination and I found it the most dificult part. I had to rely on third party solutions to accomplish it. But not before understanding how it worked ;)

Learnings

I got more comfortable with promises and asynchronous JavaScript in general, as well as with TypeScript.
